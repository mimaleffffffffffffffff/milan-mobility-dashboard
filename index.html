Hotspots

Hotspots represent areas where multiple mobility services (such as bus stops, bicycle parking, stations, and parking) are spatially concentrated. Each red point is a summarized representative location of one cluster.

Clustering basis

Hotspots are identified using a density-based clustering approach, which groups mobility points that are close to each other within a defined distance and minimum number of elements.

Dominant type

The dominant type indicates the most frequent mobility service inside the hotspot. A hotspot may still contain multiple service categories even if one type is dominant.

Diversity

Diversity represents how many different mobility service categories are present within a hotspot. Higher diversity suggests stronger multimodal mobility availability.

How to interpret

Hotspots with high size and high diversity may indicate well-connected multimodal areas, while low-diversity hotspots may suggest single-mode concentration or gaps in mobility integration.

This map is intended for exploratory analysis and should be combined with contextual layers such as population, land use, or accessibility indicators for planning decisions.
